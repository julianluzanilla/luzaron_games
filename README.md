# Luzaron Games

Luzaron Games es una PWA familiar de juegos mentales, gratuita, sin anuncios, instalable y usable offline.

URL de producción:

https://games.luzaron.uk

## Juegos iniciales

- Queens
- Sudoku
- Wordle

## Stack técnico

- Vite
- TypeScript
- HTML
- CSS
- PWA
- Service Worker
- Web App Manifest
- GitHub
- Cloudflare Pages

## Requisitos locales

Instalar:

- Git
- Node.js
- npm
- Visual Studio Code

## Instalar proyecto en una computadora nueva

```powershell
cd C:\
mkdir dev
cd dev
git clone https://github.com/julianluzanilla/luzaron_games.git
cd luzaron_games
npm install
npm run dev
```

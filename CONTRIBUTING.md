# Guía de trabajo — Luzaron Games

Esta guía define cómo trabajar en el proyecto desde diferentes computadoras.

## Regla principal

GitHub es la fuente central del proyecto.

No se debe usar Google Drive, OneDrive ni Dropbox como carpeta activa de desarrollo.

## Antes de empezar a trabajar

```powershell
cd C:\dev\luzaron_games
git status
git pull origin main
npm install
npm run dev
```
